Zodiac Algorithm Website

This website is my project for IDM231, Winter Quarter 2025-2026. I have created a site that tells visitors what Six Flags Great Adventure roller coaster they are based on their zodiac symbol. Whenever a new zodiac sign is selected, the page content updates and a sound plays.

Tech Used: HTML, CSS, JavaScript, GitHub, VSCode, Figma, Vercel
